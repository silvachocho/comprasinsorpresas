Stack: HTML + CSS + JavaScript simple. Deploy en Vercel. Analytics con Google Analytics.
Uso principal: navegador, móvil y desktop.
Etapa: Proyecto real en crecimiento, superada la fase MVP.
Prioridad: simple, útil, sin complejidad innecesaria. No monetizar aún.
